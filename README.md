"# To-Do-List" 
User story
    As a user I want to:
    
        -Sign up
        -Login
        -Loginout
        -Create to-do list
        -Edit to-do list
        -Delete to-do list
        -Complete to-do list
User model:

    name: String
    Username: String
    Password: String
    
To-Do List model:

    name: String
    complete: Boolean
    creator: UserID
Activity model:
    
    activity: String
    description: String
    due-date: String
    ListId: To-do List Id
    
 Strech Goal:
 
    Create a calender to show the due date
    Send autoamted email reminders
